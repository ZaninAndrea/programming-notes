# Chapter 1
## Choosing a distro
There are 3 main distro families:
* Fedora
    * CentOS
    * Red Hat
    * preferred by enterprises
    * `yum` packet manager
    * kernel version 3.10
* Debian
    * Ubuntu
    * focus on stability
    * largest software repository
    * kernel version 4.4 in Ubuntu 16.04 LTS
    * `apt` packet manager
* SUSE
    * openSUSE
    * kernel version 3.3
    * `zypper` packet manager
